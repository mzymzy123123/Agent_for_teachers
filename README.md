根据requirements.txt安装环境即可跑通代码，修改api调用方式即可（鼠鼠是直接把公司的模型调用账号id和key写入环境变量）
功能包括：
用户登录界面（用户分为普通教师用户和教师管理人员）
普通用户的agent功能：
1.授课视频评估（核心功能）
调用语音大模型（我这边选的硅基流动，无他，新人注册免费尔）语音转文字
调用多模态大模型评估教师外功（肢体语言、口头禅、背板、板书等）与内功（授课内容、逻辑性）并打分
针对背板时长、口头禅进行了统计，尤其是背板较长的时段列表展示，用户可以点击对应片段复盘自己的背板视频片段
同时针对板书也有截图记录可查看，最后生成评语与建议
用户可以下载完整的pdf评估文档存档
2.ppt评估功能
用户可上传制作的ppt交由调用的大模型进行三方面的评估
3.历史记录
4.教师社区
用户可一键上传自己的评课报告去社区进行交流、同时可以发布各类帖子进行社区交流

管理员的agent功能：
1.批课报告管理：可查询各教师的评课报告
2.教师社区：普通用户可以艾特管理员进行批课点评
